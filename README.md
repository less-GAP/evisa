## Local Installation

#### Install required tools and dependencies:

* [Docker](https://www.docker.com/community-edition#/download)

#### Run local development

```bash
sh dev.sh
```

* [Frontend url] (http://localhost:9100/)
* [Admin url] (http://localhost:9200/)

#### Build for production

```bash
sh build.sh
```

#### Run production

```bash
docker-compose up
```
